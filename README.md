# roulette
면접 질문을 랜덤으로 뽑기 위한 룰렛입니다.

# 요구사항
## 필수 요구사항
- [x] 텍스트창에 질문을 입력할 수 있어야 한다.
- [ ] 룰렛 버튼을 누르면 랜덤으로 저장된 내용중 하나를 선택한다.
- [ ] 선택된 문항은 다시 선택되지 않는다.
- [ ] 입력된 항목은 삭제 선택시 선택된 항목을 삭제할 수 있다.
- [ ] 항목 초기화를 누르면 저장된 전체 데이터가 삭제된다.

# 추가 요구사항
- [ ] 출력 버튼을 누르면 입력한 항목을 pdf로 출력한다.
- [ ] 저장 버튼을 누르면 로컬 스토리지에 저장한다.
- [ ] 저장된 리스트가 화면에 나타난다.
